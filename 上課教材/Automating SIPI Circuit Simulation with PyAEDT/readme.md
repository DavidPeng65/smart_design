

## 📘 課程簡介

本課程專為從事高速電路模擬的工程師設計，透過 ANSYS 提供的 Python 自動化介面 **PyAEDT**，學員將學會如何自動化進行 SIPI「信號與電池寬度」電路模擬工作，如建立網表、加入傳輸線模型、執行時域反射分析 (TDR)，以及匯出 S 參數。課程從基礎環境建立到進階自動化應用，透過實作方式快速掌握 PyAEDT 的強大能力。
